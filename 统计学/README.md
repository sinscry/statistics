	6. 统计量及其抽样分布
		* 标准正态分布表:https://www.wendangwang.com/doc/ebb144e25e8c225cb3f0f17a
		* t分布表:http://www.360doc.com/content/12/0307/17/7598058_192529468.shtml
		* 统计学n>=30为大样本，n<30为小样本是经验说法
		* 中心极限定理转换为正态分布，再转为标准正态分布求概率
		* 中心极限定理:设均值为μ，反差为σ^2（有限）的任意抽样量为n的样本，当n足够大，样本均值的抽样分布近似于均值为μ，方差为σ^2/n的正态分布
		1. 正态分布导出的重要分布
			1. X^2分布:
				* n为独立变量个数
				* E(X^2)=n
				* D(X^2)=2n
				* X^2分布的极限分布是正态分布
			2. t分布(学生氏分布):
				* X~N(0,1),Y~X^2(n),且X与Y独立则: t=X/(sqrt(Y/n))
				* t分布的诞生对小样本理论有着重要的促进作用
				* 偶函数
			3. F分布
				1. 假设变量Y与Z相互独立，且Y和Z分别服从自由度为m和n的X^2分布
					* 随机变量X: X = (Y/m)/(Z/n)=nY/mz
					* 则称X服从第一自由度为m，第二自由度为n的F分布，记为F(m,n),简记为X~F(m,n)。F分布的密度函数图为6-3
				2. 设随机变量X服从F(m,n)分布，则数学期望和方差分别为:
					* E(X)=n/(n-2),n>2
					* D(X)=2n^2(m+n-2)/(m(n-2)(n-4), n>4
			4. F分布与t分布存在以下关系
				* 如果随机变量X服从t(n)分布，则X^2服从F(1,n)的F分布，这在回归分析的回归系数显著性检验中有用
			
		
	7. 参数估计:
		* 样本估计
			1. s=sqrt(σ^2/(n-1))
			2. 平均数还是样本平均数
		* 参数估计:通过样本量和方差(标准差)以及标准正态分布表计算参数在某个值的置信空间
		* 估计方法:
			1. 点估计:用样本统计量取值直接做总体参数θ的估计值
			2. 区间估计:根据样本统计量的抽样分布可以对样本统计量与总体参数的接近程度给出一个概率度量
		* 置信空间:由样本统计量所构造的总体参数的估计区间称为置信空间
		* 注意因为总体参数是真值，所以不能说总体参数有多少概率落在某区间（客观上只有在和不在而已）
		* 大样本标准化随机变量服从标准正态分布:
			z = (平均值-μ)/(σ/sqrt(n))~N(0,1)
		* 总体均值μ在1-a置信水平下的置信空间可以写为
			平均值+-Z(a/2)*S/sqrt(n) (注意(a/2)是下标)
		* 不同情况下总体均值的区间分布:
			1. 只有小样本σ未知的情况下用平均值 +- t(a/2)*s/sqrt(n) (a/2)是下标
			2. 其他情况用 平均值 +- Z(a/2)*s/sqrt(n) (a/2)是下标
		* 区间估计参数
			1. 总体比例的区间估计置信空间: p +- Z(a/2)*sqrt(p*(1-p)/n)
			2. 总体方差的置信空间: (n-1)s^2/X^2(a/2) <= σ^2 <= (n-1)s^2/X^2(1-a/2) (其中a/2和1-a/2为X的下标)
			3. 估计总体样本量的确定: 
				1. 估计总体均值时：n=(Z(a/2))^2*σ^2/E^2 (a/2为X的下标,E为误差)
				2. 估计总体比例时：n=(Z(a/2))^2*π(1-π)/E^2
	
	8. 假设检验：
		1. 大部分：Z=(平均值-μ0)/(s/sqrt(n))
		2. 小样本无s用t分布: t=(平均值-μ0)/(s/sqrt(n)) 然后查 t(n-1) 表
		3. P值：当原假设真时得到样本观察结果出现的概率

	10. 方差分析
		* 通过检验各总体的均值是否相等来判断分类型自变量对数值型因变量是否有显著影响
		* 理论:
			* 倘若自变量对因变量无影响则组间误差只包含随机误差，无系统误差。组间误差与组内误差的方差应该很接近
			* 总误差SST(总平方和):
				1. 组内误差SSE: 水平数据内部由抽样的随机性造成的随机误差
				2. 组间误差SSA: 
					1. 随机误差：抽样本身形成的随机误差
					2. 系统误差：行业本身的系统性因素造成的系统误差
		* 例子: p241
	
	11. 一元线性回归
		* 相关关系: 变量之间存在的不确定
		* 研究方法：先绘制散点图判断变量关系，若是线性关系，可以利用相关系数判断变量关系强度
		1. 散点图：两两绘图观察
		2. 相关系数：
			* 根据样本数据计算的度量两个变量线性关系强度的统计量
			* 性质:
				1. r取值范围是[-1,1]，|r|=1时二者为函数关系
				2. r具有对称性
				3. r的数值大小与x和y的原点及尺度无关
				4. r仅仅是x与y之间线性关系的度量，不能用于描述非线性关系。因此r=0只表示二者无线性关系，并不说明变量间无任何关系，应结合散点图作出合理解释
				5. r虽然是两个变量之间线性关系的一个度量，却并不意味着x与y一定有因果关系
			* 相关关系的显著性检验:
				* 总相关系数p是未知，用样本相关系数r作为近似估计值。需要考察样本相关系数的可靠性
				* 通过假设验证来解决:p270
		* 参数的最小二乘估计（最小平方法）：
			* 注意：回归分析计算量较大，手工计算几乎不可能
			* 拟合程度判断：
				1. 判定系数：
					* 总平方和SST: ∑(y-y平均)^2
					* 回归平方和SSR: ∑(y预测-y平均)^2 
					* 残差平方和SSE∑(y-y预测)^2
					* 判定系数R^2=SSR/SST (越接近1拟合程度越好)
					* 平均标准误差Se=sqrt(SSE/(n-2))
		
		
	12. 多元线性回归
		* 描述因变量y如何依赖多个自变量和误差项的方程
		* 算法：依然是最小二乘法
		* 判定系数：
			1. SST = SSR+SSE
				* SST = ∑(y-y平均)^2
				* SSR = ∑（y预测-y平均)^2
				* SSE = ∑(y-y预测)^2
			2. R^2 = SSR/SST = 1-SSE/SST
		* 放到机器学习那硬刚算了
	
	13. 时间序列分析和预测
		* 增长率
			* 环比增长率： G = Y(i) / Y(i-1) - 1
			* 定基增长率： G = Y(i) / Y(0) - 1
			* 平均增长率： G = (Y(n) / Y(0))^(1/n)
		* 时间序列类型和预测方法：
			1. 选择逻辑
				* 有季节性
					* 季节性预测法
					* 季节多元回归模型
					* 季节自回归模型
					* 时间序列分解
				* 无趋势 & 无季节性：
						* 平滑法预测
						* 移动平均法
						* 简单平均法
						* 指数平均法
				* 有趋势 & 无季节性：
					* 趋势预测法
					* 线性趋势推测
					* 非线性趋势推测
					* 自回归预测模型
			2. 方法介绍：
				1. 简单平均法：
					* F(t+1) = ∑Yi/t
				2. 移动平均法
					* F(t+1) = (Y(t-k+1)+...+Y(t)) / k
				3. 指数平均法
					* F(t+1) = aYi + (1-a)Ft
					
		* 评估预测方法
			* Yi是观测值，Fi为预测值
			1. 平均误差ME = ∑（Yi-Fi)/n
			2. 平均绝对误差MAD = ∑|Yi-Fi|/n
			3. 均方误差MSE = ∑（Yi-Fi)^2/n
			4. 平均百分比误差MPE = ∑(Yi-Fi)/n*100
			5. 平均绝对百分比误差MAPE = ∑|Yi-Fi|/n*100
